## run

```
npm run start
```
.env에 mode가 dev면 개발자 도구가 붙은 상태로 실행됨

## Build

npm run build

/dist 에 실행가능한 프로그램으로 생성됨

# .env
```
BROWSER=none
mode=dev or product

```
